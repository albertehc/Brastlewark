# Brastlewark

## Description

Single page app in React to show data from an static JSON.

## Routes

| Path   | Component      | Permissions | Behavior        |
| ------ | -------------- | ----------- | --------------- |
| `/`    | HomeComponent  | public      | Landing page    |
| `/:id` | GnomeComponent | public      | Individual info |
